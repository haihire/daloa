# 로컬 개발 실행 루틴 (Docker 없이)

## 순서대로 실행

1. Redis 시작 (터미널 1)
   redis-cli shutdown
   redis-server

2. NestJS 서버 시작 (터미널 2)
   cd server
   npm run start:dev

3. Next.js 클라이언트 시작 (터미널 3)
   cd client
   npm run dev

## 참고

- MariaDB는 Windows 서비스로 자동 실행 중 (별도 실행 불필요)
- MariaDB 경로: C:\Program Files\MariaDB 10.6\bin\mysql.exe
- NestJS 첫 실행 시 loa_class_summaries 데이터 없으면 자동 크롤링 시작 (약 1~2분 소요)

## (선택) 크롤러 수동 실행

cd crawlers
