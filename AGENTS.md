## Workspace Agent Rules (Common)

이 파일은 루트 공통 원칙만 다룹니다.

영역별 상세 규칙:

- 서버: `server/AGENTS.md`
- 클라이언트: `client/AGENTS.md`

우선순위: 공통 규칙을 기본으로 적용하고, 영역별 파일의 규칙이 더 구체적일 경우 영역별 규칙을 우선 적용합니다.

공통 테스트 원칙:

- 테스트 이름과 assertion은 1:1로 대응합니다.
- 실제 런타임 호출 경로를 spy/mock 합니다.
- 목 데이터는 타입 계약을 지키고, 테스트마다 mock 상태를 복구합니다.
- 모호한 존재 확인만으로 통과시키지 않고, 동작/결과를 명시적으로 검증합니다.
- 에러 체크나 테스트 실패 분석 시에는 테스트 출력과 함께 `server/logs/`, `client/logs/`의 오늘치 `app-*.log`와 필요 시 `error-*.log`를 먼저 확인합니다.
